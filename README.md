🤖 **Chatbot RAG**

A hands-on project for building intelligent, context-aware chatbots using Retrieval-Augmented Generation (RAG) and LangChain — guided by DataCamp.

📌 **Overview**
This project walks through the end-to-end construction of a RAG-powered chatbot. Rather than relying solely on a language model's pre-trained knowledge, the chatbot retrieves relevant information from a custom knowledge base before generating a response — making it more accurate, grounded, and adaptable to specific domains.
Source: DataCamp
Project Name: Créer des Chatbots RAG
